To Install, put these folders into the following directories:

Window borders:     /usr/share/themes/MacBuntu-Sierra-dark
Desktop:                      /usr/share/themes/CGreen
Icons:                             /usr/share/icons/Pop
Mouse Pointer:          /usr/share/icons/Bibata_Pink

# To Install by Terminal
# cd into directory

sudo cp ./MacBuntu-Sierra-dark /usr/share/themes/
sudo cp ./CGreen /usr/share/themes/
sudo cp ./Pop /usr/share/icons/
sudo cp ./Bibata_Pink /usr/share/icons/
